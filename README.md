## 🚀 Run Locally

Clone the project

```bash
git clone https://github.com/Manjit-Kumar-Mahato/developer-portfolio.git
```

Go to the project directory

```bash
cd developer-portfolio
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
